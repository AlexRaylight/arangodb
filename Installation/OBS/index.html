Add repository and install manually

For xUbuntu 12.04 run the following as root:

echo 'deb http://www.arangodb.org/repositories/stable/repositories/xUbuntu_12.04/ /' >> /etc/apt/sources.list.d/arangodb.list 
apt-get update
apt-get install arangodb

You can add the repository key to apt like this: 

wget http://www.arangodb.org/repositories/stable/repositories/xUbuntu_12.04/Release.key
apt-key add - < Release.key
