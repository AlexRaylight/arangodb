ArangoDB (1.1.0)

My colleagues and I started developing a document-store, which focuses on durability of the data taking advantage of new technologies like SSD, support for graph and geo algorithms needed in social networks, ease of use for the developer and minimal effort to operate for the administrator. The design goals are described here.

The ArangoDB consists of a server, a separate shell, which allows you to administrate the server, and a set of client API for various languages.

Warning:
The documentation is still far from complete. Please contact us, if you have any questions.
